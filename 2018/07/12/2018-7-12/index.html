git工作：

　　　四个分区：
　　　Workspace：工作区
　　　Index / Stage：暂存区
　　　Repository：仓库区（或本地仓库）
　　　Remote：远程仓库

步骤：

  1. 建设一个仓库
  2. 邀请开发者加入
    3.将文件下拉到自己的分支工作 
    git init:初始化本地版本库
git clone (ssh):克隆远程版本库
git pull origin master:将远程库的项目下载到本地快速合并
git checkout -b (name)：创建新的分支
  3. 在自己的分支进行操作
  4. 合并上传
    git add .: 跟踪所有文件
git commit -m "注释":提交最后一次修改的文件到栈区
git checkout master:切换分支；
git merge （name）:将分支合并到主干
git push origin master:上传到远程库
